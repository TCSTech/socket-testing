# Sockets Testing

## Node

### Installation

1. Clone repository into new directory
2. `cd` into webroot 
3. Run `node install`
4. Run `bower install`
5. cd into socket.io 
6. Run `node install`
7. Run `node app` to start the node server
8. Open browser to http://localhost:3000 to see test app
